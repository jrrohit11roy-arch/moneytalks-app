# RR Visual Trading App

This is the complete local app for your CCL and FFL share market.

## What It Does

- Users create/open an account with name, Indian mobile number, location, and joining code `123456788`.
- Users request `cc` coins from admin.
- Admin unlocks the panel with password `rohit20089`.
- Admin controls live CCL and FFL prices.
- Admin approves or rejects coin requests.
- Users buy and sell CCL/FFL shares using `cc` coins.
- Admin can see every user, phone number, location, coins, shares, and trades.
- Data is saved in `data.json`.

## How To Open

Double-click:

```text
start-app.bat
```

Keep that window open, then open:

```text
http://localhost:3000
```

## Different Locations

This app runs on your computer. For users in different locations to use it on their phones, the computer/server must be online and reachable from them, or the app must be hosted on a public server.
